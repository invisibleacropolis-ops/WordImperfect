Core Features
Text Formatting: Change font, size, and color; apply bold, italic, and underline styles; and align text.
Paragraph Formatting: Adjust indents, create bulleted and numbered lists, and set margins.
Editing: Cut, copy, paste, find, and replace text.
Object Insertion: Insert images, tables, and other objects from other applications.
File Management: Save documents in Rich Text Format (.rtf), Plain Text (.txt), and Word Document (.docx) formats.
